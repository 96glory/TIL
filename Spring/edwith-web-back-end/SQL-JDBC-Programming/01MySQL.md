# 1. intro

## DB와 DBMS의 차이점
도서관에 있는 책들이 DB라 한다면, 도서 정보를 찾아주는 매개체를 DBMS라고 비유할 수 있다.

## DB의 정의
* a set of data
* 여러 응용 시스템들의 통합된 정보들을 저장하여 운영할 수 있는 공용 데이터의 집합
* 효율적으로 저장, 검색, 갱신할 수 있도록 데이터 집합들끼리 연관시키고 조직화되어야 한다.
